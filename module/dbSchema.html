# 数据库架构模块

实现在线查看当前默认数据源中表结构及导出。

# 使用方法

项目文件 pom.xml 中添加依赖：

<!--XCI数据库架构模块-->
<dependency>
    <groupId>com.github.lvyanyang</groupId>
    <artifactId>xci-framework-db-schema</artifactId>
    <version>${xci.framework.version}</version>
</dependency>
