# 文件上传模块

基于SpringMVC自主实现的轻量级文件上传、下载以及文件查询服务。

本模块支持三种类型的文件上传下载：

  • 常规文件上传下载：不能指定上传文件的具体文件名，可以指定上传目录，文件名自动生成，通过返回值传给客户端；可以通过Url地址直接访问；通过下载接口获取；

  • 数据文件上传下载：必须指定传文件的具体文件名，文件名由客户端生成；不能通过url直接访问，只能通过下载接口获取；

  • 客户端更新包上传下载：不能指定上传文件的具体文件名称，上传时必须指定密码；可以通过Url地址直接访问；

# 功能特性

  • 文件上传：支持自动命名；支持指定上传的目录；支持数据文件的上传和下载；
  • 文件下载：支持按路径下载；支持md5参数，如果md5值相同，文件则不会传输；指出url与磁盘映射配置;
  • 文件删除：支持远程删除文件；可通过参数关闭此功能；
  • 文件复制：支持远程复制文件；
  • 检查文件是否存在
  • 查询文件列表
  • 查询文件数量
  • 查询文件详细信息
  • 客户端更新包
  • 数据文件上传下载

# 使用方法

项目文件 pom.xml 中添加依赖：

<!--XCI文件上传模块-->
<dependency>
    <groupId>com.github.lvyanyang</groupId>
    <artifactId>xci-framework-upload</artifactId>
    <version>${xci.framework.version}</version>
</dependency>

# 配置参数

Spring配置参数 
# 是否启用上传，默认启用
spring.servlet.multipart.enabled=true

# 单个文件允许上传的大小，默认1MB
spring.servlet.multipart.max-file-size=1MB

# 整个请求允许上传的总大小，默认10MB
spring.servlet.multipart.max-request-size=10MB

# 如果上传文件大小达到了指定大小，将会写入到临时文件中，如果没有达到指定大小，则会在内存中缓存。默认值为0，也就是所有上传的文件都会写入到临时文件中
spring.servlet.multipart.file-size-threshold=1MB

server.tomcat.max-swallow-size=200MB

模块配置参数 
# 是否启用文件上传功能
xci.upload.enabled=true
    
# 是否启用文件上传token验证，如果启用则需要在请求header集合中添加uploadToken，主要是基于安全考虑
xci.upload.enable-token=false

# 是否启用文件删除接口
xci.upload.enable-delete=false

# 文件磁盘保存目录，需要指定绝对路径，{root}是变量，表示项目的启动路径
xci.upload.file-root=/{root}/uploads

# 文件http访问根路径，如果为空，则不允许通过http访问
xci.upload.web-root=/uploads/

# 允许上传的单个文件大小（KB）
xci.upload.max-size=204800

# 允许上传的文件扩展名（不含点）
xci.upload.allow-extensions=gif,jpg,jpeg,png,bmp,zip,rar,doc,docx,xls,xlsx,ppt,pptx,txt,pdf,repx,json,xml

#是否只允许上传图片,如果为true则allowExtensions属性会被忽略，只允许上传图片(检测内容必须是图片才能上传)
xci.upload.only-allow-image=false

# 接口调用

WebApi接口调用需要在请求 Header 集合中添加以下变量：

名称 值 说明
X-Requested-Type APIHttpRequest 接口请求标识
uploadToken 上传Token 如果开启文件Token验证

# Token校验

为了保障接口安全，可以控制是否需要在上传文件时进行文件Token校验。

  1. 开关

在配置文件 application.properties 中通过 xci.upload.enable-token=true 开启或者关闭

  1. 规则

如果开启文件Token验证，需要客户端调用接口时在请求 Header 中添加变量 uploadToken，变量的值为当前日期按此格式 *yyyy**MM***dd**** 进行格式化，然后调用加密函数。

//java实现
XCI.encrypt(XCI.formatDate(new Date(), "*yyyy**MM***dd****"));

//c#实现
EncryptHelper.AESEncrypt(DateTimeHelper.FormatDate(DateTime.Now, "*yyyy**MM***dd****"));

上传服务常用方法 
@Service
public class UploadService {
    /**
     * 上传文件
     * @param uploadFile     上传文件对象
     * @param virtualCatalog 文件虚拟目录(选填)
     * @return 返回上传文件信息
     */
    public BoolResult<UploadFileModel> upload(MultipartFile uploadFile, String virtualCatalog);

    /**
     * 下载文件
     * @param virtualFilePath 文件虚拟路径
     * @param md5             文件Md5
     */
    public void download(String virtualFilePath, String md5);

    /**
     * 上传数据文件
     * @param uploadFile      上传文件对象
     * @param virtualFilePath 虚拟文件路径
     * @return 返回上传文件信息
     */
    public BoolResult uploadDataFile(MultipartFile uploadFile, String virtualFilePath);

    /**
     * 下载数据文件
     * @param virtualFilePath 虚拟文件路径
     * @param md5             文件Md5
     */
    public void downloadDataFile(String virtualFilePath, String md5);

    /**
     * 删除文件
     * @param virtualFilePath 文件虚拟路径
     */
    public BoolResult delete(String virtualFilePath);

    /**
     * 复制文件
     * @param srcVirtualFilePath  源文件虚拟路径
     * @param destVirtualFilePath 目标文件虚拟路径
     */
    public BoolResult copy(String srcVirtualFilePath, String destVirtualFilePath);

    /**
     * 检测文件是否存在
     * @param virtualFilePath 虚拟文件路径
     */
    public BoolResult<Boolean> exists(String virtualFilePath);

    /**
     * 查询文件列表
     * @param virtualCatalog 文件虚拟目录
     */
    public BoolResult<List<FileModel>> list(String virtualCatalog);

    /**
     * 获取文件数量
     * @param virtualCatalog 文件虚拟目录
     */
    public BoolResult<Long> count(String virtualCatalog);

    /**
     * 获取文件信息
     * @param virtualFilePaths 虚拟文件路径字符串,多个路径间用逗号隔开
     */
    public BoolResult<List<UploadFileModel>> info(String virtualFilePaths);
    /**
     * 虚拟路径转为磁盘物理路径
     * @param virtualPath 文件虚拟路径
     */
    public String toUploadDiskPath(String virtualPath);

    /**
     * 虚拟路径转为http请求路径
     * @param virtualPath   文件虚拟路径
     * @param isFullSiteUrl 是否返回完整http地址
     */
    public String toUploadWebUrl(String virtualPath, Boolean isFullSiteUrl);

    /**
     * 获取自动生成的文件虚拟路径
     * @param uploadFile     上传文件对象
     * @param virtualCatalog 文件虚拟目录
     */
    private BoolResult<String> getUploadAutoSaveVirtualFilePath(MultipartFile uploadFile, String virtualCatalog);

    /**
     * 注册上传文件资源路径映射
     */
    public void registryResource(ResourceHandlerRegistry registry);
}
