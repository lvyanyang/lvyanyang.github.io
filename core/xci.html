# 核心工具类

框架针对项目中使用频率极高的功能函数以静态方法的方式统一封装到XCI类中，方便使用，下面介绍一下常用的函数。

# 字符串类

  • cleanHtml 清除Html不安全标签
  • cleanXss 清除Xss不安全内容
  • encodeHtml HTML字符转义
  • decodeHtml 还原被转义的HTML字符
  • htmlToText 提取html中的文字
  • format 格式化输出文本
  • getSpell 获取所给中文的每个汉字首字母组成首字母字符串
  • getSpell 获取所给中文的每个汉字字母字符串
  • splitToList 切分字符串到List
  • splitToArray 切分字符串到数组
  • replaceEnter 替换字符串中的回车和换行
  • pathOf 路径拼接
  • pathOfClean 路径清理

# 空判断

  • isEmpty 字符串是否为空，List，Array，Map元素长度0也表示空
  • isNotEmpty
  • isBlank 去除掉不可见字符后是否为空
  • isNotBlank

# 加密

  • fromHex 十六进制字符串转为字节数组
  • toHex 字节数组转为十六进制字符串
  • toBase64 将字节码转换成base64文本
  • toBase64Str 将文本转换成base64文本
  • fromBase64 将base64转换成字节码
  • fromBase64Str 将base64转换成文本
  • encrypt 字符串AES对称加密
  • decrypt 字符串AES对称解密

# 逻辑判断

  • ifBlankAction 对象为空或者空字符串时执行语句
  • ifNotBlankAction 对象不为空时执行语句
  • ifNullAction 对象为空时执行语句
  • ifNotNullAction 对象为空时执行语句
  • ifTrueAction 测试条件为true时执行语句
  • ifFalseAction 测试条件为false时执行语句
  • ifAction 根据测试条件执行语句

# ID生成器

  • guid 生成普通 Guid
  • objectId 创建MongoDB ID生成策略实现
  • nextId 获取新唯一编号（18位数值）
  • nextIdStr 获取新唯一编号（18位数值）字符串

# 异常处理

  • ifTrueThrow 如果测试条件为 true,则抛出应用程序异常
  • ifTrueThrow 如果测试条件为 true,则抛出应用程序异常
  • ifFalseThrow 如果测试条件为 false,则抛出应用程序异常
  • ifFalseThrow 如果测试条件为 false,则抛出应用程序异常
  • ifNullThrow 如果测试对象为null,则抛出应用程序异常
  • ifNotNullThrow 如果测试对象为null,则抛出应用程序异常
  • ifBlankThrow 如果测试对象为null或者内容为空,则抛出应用程序异常
  • ifNotBlankThrow 如果测试对象为null或者内容为空,则抛出应用程序异常
  • throwException 抛出应用程序异常

抛出的异常类 BusinessException

# 文件操作

  • buildRootFile 构造根目录文件对象
  • buildRootFilePath 构造根目录文件路径
  • readFileByMd5 读取文件检测 md5
  • readFileByMd5 读取文件检测 md5
  • writeFileByUtf8 写入文件
  • deleteFile 删除文件
  • getFileExtName 获得文件的扩展名，扩展名不带.
  • getFileMainName 返回主文件名
  • getWebRoot 获取Web项目下的WebRoot路径

# 日期处理

  • now() 获取当前日期
  • formatDate 格式化日期(默认格式yyyy-MM-dd)
  • formatDateHasMinute 格式化日期字符串(默认格式 yyyy-MM-dd HH:mm)
  • formatDateTime 格式化日期(默认格式 yyyy-MM-dd HH:mm:ss)
  • formatTime 获取格式化的时间字符串(默认格式 HH:mm:ss)
  • formatDate 格式化日期
  • dateToLocalDateTime 日期对象转为本地日期时间对象
  • dateToLocalDate 日期对象转为本地日期对象
  • dateToLocalTime 日期对象转为本地时间对象
  • localDateToDate 本地日期对象转为日期对象
  • localDateTimeToDate 本地日期时间对象转为日期对象
  • localDateTimeToDate 本地日期和本地时间对象转为日期对象
  • isExpired 测试检测时间是否在指定的开始时间和结束时间之内,精度到毫秒单位,如果不在范围内返回true

# 集合处理

  • updateListItem 更新列表元素
  • listIndexOf 根据条件获取元素索引号
  • indexOf 在一个一维数组中搜索指定对象
  • pageList 列表分页排序

# 类型转换

  • toChar 转换为字符
  • toByte 转换为byte
  • toShort 转换为Short
  • toNumber 转换为Number
  • toInt 转换为int
  • toLong 转换为long
  • toDouble 转换为double
  • toFloat 转换为Float
  • toBool 转换为boolean
  • toEnum 转换为Enum对象
  • toBigInteger 转换为BigInteger
  • toBigDecimal 转换为BigDecimal
  • toStr 转换为字符串
  • toIntArray 转换为Integer数组
  • toLongArray 转换为Long数组
  • toStrArray 转换为String数组

# Tree操作

  • buildTreePath 生成路径字符串
  • buildNewTreePath 获取新建节点排序路径
  • updateNodeDrag 更新拖拽节点
  • deleteTree 删除树形结构
  • buildFilterTreePredicate 动态构建Tree过滤条件
  • filterTree 树形结构数据过滤(自动关联出结果的所有上级)
  • getDeleteChildrenIds 获取待删除的主键集合(包含指定Id字符串中所有的下级主键)
  • getTreeParents 获取指定节点的所有父节点
  • getTreeChildren 获取指定节点的所有子节点
  • removeTreeChildren 移除子节点
  • hasChildren 检查指定节点Id是否存在子节点

# Json操作

  • toJsonString 对象转Json字符串
  • toJsonObject Json字符串转对象
  • toJsonListObject Json字符串转集合对象
  • toJsonMapObject Json字符串转Map对象
  • loadJsonList 加载 Json 文件数据
  • saveJsonList 保存 Json 文件数据

# Cookie操作

  • getCookies 获取Cookie数组
  • getCookie 获取单个Cookie
  • setCookie 设置 Cookie
  • removeCookie 删除cookie

# Jwt

  • createJwt 生成jwt令牌
  • parseJwt 解析jwt
  • getTokenId 获取 TokenId

# 缓存

  • getCacheValue 获取缓存对象值
  • getCacheValueList 获取缓存对象集合
  • cacheContains 检查缓存容器中是否存在指定名称的对象

# Spring容器

  • getBean 通过name、class获取Bean对象
  • getBeansOfTypeList 获取接口实现类集合
  • getBeansOfType 获取接口实现类Map
  • containsBean 通过name、class测试Bean对象是否存在，则返回true
  • isSingleton 判断Bean对象是否是单例
  • getSpringType 根据名称获取Bean对象类型
  • getCurrentProxy 获取Aop代理对象
  • registerBean 主动向Spring容器中注册bean

# 其他

  • startWatch 开始计时监控
  • stopWatch 停止计时监控
  • plusEndDateTime 处理结束日期,结束日期增加一天
  • getDefaultDatabaseId 获取默认数据源的数据库标识符
  • trimMapValue 去除map中字符串值两边空格
  • download 文件下载
  • getRequest 获取Request对象
  • getResponse 获取Response对象
  • getSession 获取Session对象
  • getHostIP 获取服务器IP
  • getHostName 获取服务器主机名
  • getClientIP 获取客户端IP
  • getIPLocation 根据ip地址解析IP地址地理位置
  • getRequestBrowserOsInfo 获取当前请求浏览器操作系统IP地址信息
  • forEachIds 分割主键字符串,并进行for循环
  • buildFilterPredicate 动态构建过滤条件
  • filterList 过滤列表
