# 基础工具库

Hutool (opens new window)是一个小而全的Java工具类库，通过静态方法封装，降低相关API的学习成本，提高工作效率。

# 错误返回方式

服务层方法在做业务处理时，可能做大量的业务判断，如果需要操作终止返回，有两种方式：

  1. return BoolResult.fail(msg) 直接返回BoolResult对象
  2. throw new BusinessException(msg) 抛出异常

区别：如果返回时需要回滚事务，则必须抛出异常，并且方法加@Transactional(rollbackFor = Exception.class)注解，否则直接返回BoolResult对象。
