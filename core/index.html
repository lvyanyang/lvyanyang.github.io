# 介绍

西交投行业系统开发平台是一个交通行业管理系统快速开发平台，基于Java平台构建，采用模块化设计。主要目的让开发者专注业务，降低技术难度，从而节省人力成本，缩短项目周期，提高软件安全质量。

# 技术选型

  • Jdk 11+
  • Apache Maven 3.6+
  • Spring Boot 2.2.6.RELEASE+
  • Spring Framework 5.2.5.RELEASE+
  • Apache MyBatis 3.5+
  • Freemarker 2.3+
