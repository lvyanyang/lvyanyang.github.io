# 快速开始

# 1. 新建SpringBoot项目

根据向导新建标准 SpringBoot 项目。

提醒

JDK版本：11
SpringBoot版本: 2.2.6.RELEASE
SpringBoot官方地址: https://start.aliyun.com
国内镜像: https://start.spring.io

添加核心包依赖

<dependency>
    <groupId>com.github.lvyanyang</groupId>
    <artifactId>xci-framework-core</artifactId>
    <version>${xci.framework.version}</version>
</dependency>

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"

    <!--设置Maven依赖库公司私服镜像以及国内阿里云私服镜像-->

            <id>private</id>

                <enabled>true</enabled>
                <checksumPolicy>warn</checksumPolicy>
            </releases>
            <snapshots>
                <enabled>true</enabled>
                <updatePolicy>always</updatePolicy>
            </snapshots>
        </repository>
        <!--阿里云私服镜像-->
        <repository>
            <id>aliyun</id>
            <url>https://maven.aliyun.com/repository/public</url>
            <releases>
                <enabled>true</enabled>
            </releases>
            <snapshots>
                <enabled>true</enabled>
            </snapshots>
        </repository>
    </repositories>
    
    <!--设置Maven插件服务器镜像-->
    <pluginRepositories>
        <pluginRepository>
            <id>alimaven spring plugin</id>
            <name>alimaven spring plugin</name>
            <url>https://maven.aliyun.com/repository/spring-plugin</url>
        </pluginRepository>
    </pluginRepositories>
    
    <build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
    </build>
</project>

# 2. 配置启动类

  1. 启动类添加注解 @PropertySource("classpath:xci.properties")，导入应用程序基础配置。
  2. 启动类继承 BaseApplication，或者实现 IApplication接口均可。
  3. 如果加载了xci-framework-sys模块，需要继承SysApplication。
启动类 
/**
 * 应用程序启动类
 **/
@SpringBootApplication
@PropertySource("classpath:xci.properties")
// @EnableCaching
// @EnableTransactionManagement
// @MapperScan(basePackages = {"com.xci.module.om.dao"})
public class TestApplication extends BaseApplication {
    public static void main(String[] args) {
        SpringApplication.run(TestApplication.class, args);
    }
}

# 4. 添加模块

xci-framework-core是核心模块，只包括基础功能，要使用其他功能，需要添加模块依赖，目前已有模块：

  • xci-framework-core 核心模块，主要实现了数据库、日志、校验、导入导出、文档等应用基础功能。
  • xci-framework-upload 文件上传模块，主要实现了文件的上传、下载以及文件查询服务。
  • xci-framework-wechat 微信公众号开发模块，主要实现了微信公众号的接口开发与调用封装。
  • xci-framework-quartz 定时任务模块，主要实现了在线添加、修改、删除任务，控制任务启用、停用以及查看任务执行日志。
  • xci-framework-db-schema 数据库架构模块，主要实现在线查看表结构及导出。
  • xci-framework-code-generate 代码生成模块，主要实现根据数据库表、模板批量在线生成、导出代码。
  • xci-framework-sys 系统权限模块，主要实现用户、角色、机构、参数、字典、日志等基础功能及权限管理（业务实现）。
  • xci-framework-sys-ui 系统权限UI模块,主要实现了用户、角色、机构、参数、字典、日志等基础功能及权限管理（UI界面，基于jQuery+bootstrap实现）。

要添加模块到项目中，需要在 pom.xml 添加模块依赖。

查看代码 
<dependencies>
    <!--XCI上传模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-upload</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>

    <!--XCI数据库架构模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-db-schema</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>

    <!--XCI代码生成模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-code-generate</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>

    <!--XCI定时任务模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-quartz</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>

    <!--XCI系统权限模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-sys</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>

    <!--XCI系统权限UI模块-->
    <dependency>
        <groupId>com.github.lvyanyang</groupId>
        <artifactId>xci-framework-sys-ui</artifactId>
        <version>${xci.framework.version}</version>
    </dependency>
</dependencies>

# 5. 项目文档

接口文档：http://localhost:54321/doc.html (opens new window)

数据库文档(加载xci-framework-db-schema模块)：http://localhost:54321/db.html (opens new window)

代码生成(加载xci-framework-code-generate模块)：http://localhost:54321/gen.html (opens new window)

以上三个文档采用相同的过滤器进行安全验证，只要有一个验证通过即可，配置文件中的设置方式，列出的配置是默认值。

#是否启用文档权限控制
xci.swagger.auth=true
#文档访问账号
xci.swagger.user-name=xci
#文档访问密码
xci.swagger.password=xci
